# Docker Shishmakov Mikhail Homework 2 — Variant 1+2

## Что делает проект

Это простое веб-приложение на Flask, подготовленное для практического задания по Docker.  
Приложение запускается в Docker-контейнере, открывается в браузере по адресу `http://localhost:8080` и показывает, что контейнер успешно работает.  
Также приложение использует переменные окружения для настройки порта и режима отладки.

## Какой вариант задания выбран и почему
Выбран **вариант 2 — «Веб доступен снаружи»**.  
Этот вариант позволяет получить максимальное количество баллов, потому что включает в себя все требования варианта 1 и дополнительно проверяет:
- проброс порта наружу через `-p`
- доступность приложения из браузера
- запуск приложения на `0.0.0.0`, а не на `localhost`
- использование `ENV`-переменных для конфигурации

## Структура проектa
Все основные файлы находятся в корне репозитория:

- `app.py` — Flask-приложение
- `requirements.txt` — Python-зависимости
- `Dockerfile` — инструкция сборки Docker-образа
- `.dockerignore` — исключения для Docker-сборки
- `README.md` — инструкция по запуску проекта

## Как запустить проект
### 1. Клонировать репозиторий

```bash
git clone https://github.com/mikhailshish/devOps_homework2.git
cd devOps_homework2
### 2. Собрать Docker-образ
docker build -t my-web .
### 3. Запустить контейнер
docker run -d -p 8080:8080 --name app my-web
### 4. Открыть приложение в браузере

### Открыть в браузере:
http://localhost:8080
### Проверка работы контейнера
### Посмотреть логи контейнера
docker logs app
### Проверить health endpoint
curl http://localhost:8080/health

### Ожидаемый ответ: {"app":"Docker Homework App","status":"ok"}

### Как остановить и удалить контейнер
docker stop app && docker rm app
