# Docker Variant 2 Homework Шишмаков Михаил 

## Что делает проект

Это простое веб-приложение на Flask, подготовленное для практического задания по Docker.
Оно запускается в контейнере, открывается в браузере и показывает, что приложение работает, а также отображает текущие настройки PORT и DEBUG.

## Какой вариант задания я выбрал

Выбрал вариант 2 — «Веб доступен снаружи».
Ну потому что вариант позволяет получить максимальное количество баллов плюс дополнительно проверяет проброс порта, доступность приложения из браузера, использование 0.0.0.0 и ENV-переменных.
